Add `projectName` to the context of `snapshotRootDir` to allow users to customize the snapshot root directory based on the project name.
This in useful when you use multiple instances in a Vitest project.

The `projectName` is generated by Vitest (e.g. `vpv:pw (chromium)` where `vpv:pw` is the project name you defined and `chromium` is the browser name).

```ts
export default defineConfig({
	test: {
		name: 'vpv:pw',
		browser: {
			instances: [
				{ browser: 'chromium' }
			]
		}
	}
})
```

If you specify the instance name, the `projectName` will be the instance name.

The `projectName` will be `HD` in the following example.

```ts
export default defineConfig({
	test: {
		name: 'vpv:pw',
		browser: {
			instances: [
				{ browser: 'chromium', name: 'HD' }
			]
		}
	}
})
```

Also added `rootDir` to the context of `snapshotRootDir` to make the snapshot root directory more transparent.
