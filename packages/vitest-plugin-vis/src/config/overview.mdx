# Configuration

Configuration refers to configuring [vitest-plugin-vis] in your `vitest.config.ts` file.

The API is provided under `vitest-plugin-vis/config`.

As seen in the [Overview](../overview.mdx#basic-configuration),
the main API is the `vis` plugin:

```ts
import { vis } from 'vitest-plugin-vis/config'

export default defineConfig({
	plugins: [vis()],
})
```
