# Installation

[storybook-addon-vis][storybook-addon-vis] works with [Storybook Test addon][storybook-test-addon],
so naturally you need to have it setup first.

After that, you can install [storybook-addon-vis][storybook-addon-vis] with your package manager of choice:

```sh
npm install --save-dev storybook-addon-vis
```
```sh
pnpm add --save-dev storybook-addon-vis
```
```sh
yarn add --save-dev storybook-addon-vis
```

Next, you need to setup [storybook-addon-vis][storybook-addon-vis] for both [Storybook] and [Vitest].

[storybook-addon-vis]: https://www.npmjs.com/package/storybook-addon-vis
[storybook-test-addon]: https://storybook.js.org/docs/writing-tests/test-addon
[Storybook]: https://storybook.js.org/
[Vitest]: https://vitest.dev/
