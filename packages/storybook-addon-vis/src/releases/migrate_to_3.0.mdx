# Migrating to 3.0

[storybook-addon-vis][storybook-addon-vis] 3.0.0 supports Storybook 10.

The API remains the same as v2.0.0.

If you are migrating to use CSF Next, follow the [Basic Setup for CSF Next](?path=/docs/overview--docs#basic-setup-for-csf-next) guide.
